Componentes

Los componentes son el componente principal de las aplicaciones de Angular. Cada componente consta de:
Una plantilla HTML que declara lo que se representa en la página
Una clase de TypeScript que define el comportamiento.
Un selector de CSS que define cómo se usa el componente en una plantilla
Opcionalmente, estilos CSS aplicados a la plantilla.
Este tema describe cómo crear y configurar un componente Angular.

requisitos previos

Para crear un componente, verifique que haya cumplido con los siguientes requisitos previos:
Instale la CLI angular.
Cree un espacio de trabajo angular con aplicación inicial. Si no tiene un proyecto, cree uno usando ng new (project-name), donde (project-name) es el nombre de su aplicación Angular.

Creando un componente

La mejor manera de crear un componente es con Angular CLI. También puede crear un componente manualmente.

Creación de un componente usando la CLI de Angular

Para crear un componente usando la CLI de Angular:
Desde una ventana de terminal, navegue hasta el directorio que contiene su aplicación. Ejecute el comando ng generar componente (nombre-componente), donde (nombre-componente) es el nombre de su nuevo componente.

De forma predeterminada, este comando crea lo siguiente:
Una carpeta con el nombre del componente Un archivo de componente, (nombre-componente).component.ts
Un archivo de plantilla, (nombre-componente).component.html
Un archivo CSS, (nombre-componente).component.css
Un archivo de especificación de prueba, (nombre-componente).component.spec.ts
Donde (component-name) es el nombre de su componente.
